# Player UI

## Запуск

1. Убедитесь, что Host запустил сервер и сообщил IP/порт.
2. Откройте Player UI в браузере:

```
http://HOST_IP:8000/player
```

Замените `HOST_IP` и порт на данные, которые дал ведущий.

## Паринг по PIN

1. Получите PIN от ведущего.
2. В Player UI введите PIN, чтобы получить player token.
3. После успешного паринга токен сохранится в браузере и будет использоваться автоматически.

## Основные сценарии

- **Подключение к кампании**: открыть Player UI и пройти паринг по PIN.
- **Просмотр данных**: следить за листом персонажа, инвентарём, квестами и сообщениями.
- **Взаимодействие**: отправлять допустимые запросы (например, заявки в друзья или сообщения), если Host их разрешил.

## FAQ

**Что делать, если PIN не подходит?**
Проверьте, что PIN введён без пробелов и соответствует текущему PIN Host.

**Нужно ли заново вводить PIN при перезагрузке?**
Нет, токен хранится в localStorage, пока вы не очистите данные браузера.

**Можно ли подключаться с телефона?**
Да, если устройство находится в одной сети и может открыть адрес сервера.
